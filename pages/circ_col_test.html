Hand Calculation

Ag = π/4 × 500² = 196,349.5 mm² Ast = 8 × π/4 × 20² = 2,513.3 mm² ρg = 2513.3 / 196349.5 = 1.280 % fcd = fck / γc = 25 / 1.50 = 16.667 MPa fyd = fyk / γs = 500 / 1.15 = 434.783 MPa EC2 stress block: λ = 0.80, η = 1.00 (fck ≤ 50 MPa) fc_eff = η × fcd = 16.667 MPa N₀ = fc_eff×(Ag−Ast) + fyd×Ast = 16.667 × (196349.5 − 2513.3) + 434.783 × 2513.3 = 16.667 × 193,836.2 + 434.783 × 2513.3 = 3,230,769 + 1,093,172 = 4,323,941 N → 4,324 kN Nt (pure tension) = −fyd × Ast = −434.783 × 2513.3 = −1,093,172 N → −1,093 kN Balanced neutral axis depth (εcu=0.0035, εyd=fyd/Es=434.783/200000=0.002174): c_bal = εcu/(εcu+εyd) × D = 0.0035/(0.0035+0.002174) × 500 = 0.617 × 500 = 308.7 mm

Tolerance: Results within ±0.5% are considered exact (numerical integration rounding). Results within ±1.0% are acceptable (due to fiber strip discretization in moment capacity). Results beyond ±1.0% indicate a bug.

Known limitation: The column.html engine uses the tied column factor (0.80) for all circular columns. ACI 318 §22.4.2.1 permits factor 0.85 for spirally reinforced columns (φ=0.75). Spiral column results should be verified separately or the factor updated in buildCurveCirc().
